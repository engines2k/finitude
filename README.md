# Finitude

## Feature milestones
- [x] Limit subscriptions feed by video publish date (list view)
- [x] Remove feed continuation from subscriptions (list view)
- [ ] Limit subscriptions feed by video published (grid / all views)
- [ ] Remove feed continuation scroll from subscriptions (grid / all views)
- [ ] Popup for adjusting date for video filtering
- [ ] Limit videos in recommendation feed to a constant amount
